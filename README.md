# newton-fractal-p5

Newton fractal with p5.js

## Development

Requires [Node.js](https://nodejs.org/)

```bash
# clone this repository
$ git clone https://github.com/H1rono/newton-fractal-p5
# move into repository
$ cd newton-fractal-p5
# install dependencies
$ npm i
# run server
$ npm run dev
```
